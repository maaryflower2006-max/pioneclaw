# 记忆管理

帮助用户保存重要信息到长期记忆中，并在需要时检索出来。

## 使用场景

- 用户要求"记住"某些偏好、配置或事实
- 用户提到之前讨论过的内容，需要回忆
- 跨会话保持上下文连续性

## 记忆操作

### 保存记忆

当用户说"记住..."或"帮我记下..."时：
1. 提取关键信息
2. 分类标记（如：偏好、配置、事实、待办）
3. 调用记忆存储工具保存

### 检索记忆

当用户提到之前的内容或需要上下文时：
1. 根据关键词搜索相关记忆
2. 按相关度排序返回
3. 简要总结记忆内容

### 记忆分类

- **偏好**: 用户的个人喜好和设置
- **配置**: 项目或环境的配置信息
- **事实**: 重要的事实和数据
- **待办**: 需要后续跟进的事项

## 重要原则

- 只记忆用户明确要求或显然重要的信息
- 不记忆敏感信息（密码、密钥、个人隐私）
- 记忆检索时说明信息来源和时间
- 定期提醒用户清理过期记忆
